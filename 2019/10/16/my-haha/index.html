第一次使用如此奇妙的东西，甚是开心，就记录下来。

Node.js

我查阅资料得知Node.js是一个运行环境或者是一个JS语言解释器，Node.js的出现使JS可以在服务器，我们自己的笔记本上运行。Node.js使用了一个事件驱动，非阻塞式1/0的模型，使其质轻又高效。Node.js的包管理器是NPM——全球最大的的形源库生态系统。

Git

Git是开源的分布式版本控制系统，用于高效敏捷的处理项目。我们的网站在本地搭建好后，需要使用Git同步到GitHub上。设置GitHub密匙的原因是GitHub要求推送的代码是合法的，所以每次都要输入账号密码验证推送的用户是否合法，为了省去密码验证的过程，就设置了ssh。每当推送的时候，git会匹配我们的私匙与GitHub上的公匙是否配对。若配对，则是合法的。

Markdown

这是一个可以使用普通文本编辑器编写的标记语言，通过简单的标记语法使普通文本内容具有一顶的格式。它的语法简洁明了，容易掌握。而且还可在文章中添加音乐，直接把音乐的外链HTML插入文章中。
